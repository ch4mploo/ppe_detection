# Model Descriptions

## 1. Model and Data Source
The source model is from Ultralytics YOLO26 pre-trained nano model (yolo26n).

It is custom trained with Construction-PPE dataset, also provided by Ultralytics.

## 2. Model Format
The model is prepared in 3 formats:
1. Original Pytorch format (.pt) 
2. ONNX format (.onnx)
3. NCNN format (_ncnn_model) for Raspberry Pi
